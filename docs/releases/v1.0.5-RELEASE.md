# Release Notes - v1.0.5: Terminal Livestream & MCP Core Setup

**Release Date:** April 27, 2026

## 🎯 Overview

v1.0.5 introduces the **Claude Code Terminal Livestream Component** — a professional, real-time terminal output visualization widget for Claude Code's web interface, plus a complete **MCP Core Initialization System** with automated skill discovery and registry generation.

This release enables developers to visualize MCP setup processes and skill registration in real-time, making installation and configuration transparent and educational.

---

## ✨ What's New

### 🎬 Terminal Livestream Component (NEW)

A production-ready React component that displays real-time terminal output in Claude Code:

#### Features
- ✅ **Real-time log streaming** with WebSocket support
- ✅ **Color-coded output** (info, success, warning, error, command)
- ✅ **Interactive controls** — Copy, download, clear logs
- ✅ **Live/pause toggle** — Freeze log snapshots by ID (preserves view on array rotation)
- ✅ **Educational licensing** — UI badge and license enforcement
- ✅ **Accessibility** — WCAG AA compliant, keyboard navigation
- ✅ **Responsive design** — Works on mobile, tablet, and desktop
- ✅ **Branding** — Fused Gaming magenta/purple theme

#### Component Files
- `packages/web/components/TerminalLivestream.tsx` (350 lines)
- `packages/web/hooks/useTerminalLivestream.ts` (React hook with WebSocket integration)
- `packages/web/examples/TerminalLivestream.example.tsx` (10 complete examples)

#### Integration & Deployment Docs
